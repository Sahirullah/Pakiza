# Modern Portfolio Website

A beautiful, animated portfolio website built with React, Vite, and Framer Motion.

## Features

✨ **Smooth Animations** - Framer Motion animations on scroll and hover
🎨 **Modern Design** - Clean, professional design with gradient backgrounds
📱 **Responsive** - Fully responsive on all devices
⚡ **Fast** - Built with Vite for lightning-fast development
🎯 **Smooth Scrolling** - Native smooth scroll navigation

## Sections

- **Hero** - Eye-catching introduction with floating animations
- **About** - About section with stats
- **Skills** - Animated skill cards with progress bars
- **Projects** - Project showcase with hover effects
- **Experience** - Timeline of work experience
- **Certifications** - Professional certifications
- **Testimonials** - Client testimonials with ratings
- **Contact** - Contact form and information
- **Footer** - Social links and navigation

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build

```bash
npm run build
```

## Customization

Edit the component files in `src/components/` to customize:
- Colors in CSS files
- Content in JSX files
- Animation timings in component variants

## Technologies Used

- React 19
- Vite 5
- Framer Motion 10
- CSS3 with animations
- Intersection Observer API
